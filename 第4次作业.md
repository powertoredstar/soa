## 一、功能说明

程序类 `soa.Homework4`（`eclipse-workspace/soa/src/soa/Homework4.java`）完成两项功能：

1. **输入流**：读取第三次作业生成的 **`homework3_output.txt`**（UTF-8，首行 `int`、次行学号），将结果输出到控制台。流链：**`FileInputStream` → `BufferedInputStream` → `InputStreamReader(UTF-8)` → `BufferedReader`**。
2. **简单运算器**：使用 **`Scanner`**、`System.in/out`、**`StringBuilder`** 拼接结果行，支持 **`+ - * /`** 与 **`q`** 退出；除法为整数除法，除数为 0 时提示错误。

说明：控制台提示当前为 **英文**，以避免 Windows 默认 **GBK** 源编码下的编译报错。
